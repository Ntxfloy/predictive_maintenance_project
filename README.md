# Проект: Предиктивное обслуживание промышленного оборудования (ВКР)

Данный проект разработан в рамках выпускной квалификационной работы (ВКР) по направлению **«Искусственный интеллект в машиностроении»**. 
Цель проекта — создание интеллектуальной системы на базе машинного обучения, которая анализирует телеметрию с датчиков оборудования в режиме реального времени и прогнозирует вероятность аварийного останова (Machine failure).

Результаты оформлены в виде интерактивного многостраничного веб-приложения на **Streamlit** с визуализацией данных, сравнением моделей и модулем симуляции датчиков.

## 📊 Описание датасета и признаков
В качестве данных используется эталонный датасет **AI4I 2020 Predictive Maintenance Dataset** (UCI Machine Learning Repository, ID: 601), содержащий 10 000 записей со следующими признаками датчиков:

1. **Type**: Качество обрабатываемого продукта (L — низкое, M — среднее, H — высокое).
2. **Air temperature [K]**: Температура воздуха в помещении (нормализована около 300 K).
3. **Process temperature [K]**: Температура процесса обработки детали (нормализована около 310 K).
4. **Rotational speed [rpm]**: Скорость вращения шпинделя станка.
5. **Torque [Nm]**: Крутящий момент на валу.
6. **Tool wear [min]**: Время износа инструмента.
7. **Machine failure** (целевая переменная): Бинарный флаг отказа станка (0 — нормальная работа, 1 — аварийный отказ).

## 🤖 Результаты машинного обучения
Для решения задачи классификации были обучены и сопоставлены три модели: **Логистическая регрессия**, **Случайный лес (Random Forest)** и **Градиентный бустинг (XGBoost)**.

Показатели моделей на отложенной тестовой выборке (20%):

| Модель | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
| :--- | :---: | :---: | :---: | :---: | :---: |
| **Logistic Regression** | 0.8420 | 0.1650 | 0.8088 | 0.2743 | 0.9105 |
| **Random Forest** | 0.9855 | 0.7714 | 0.7941 | 0.7826 | 0.9782 |
| **XGBoost (Best)** | **0.9870** | **0.7595** | **0.8824** | **0.8163** | **0.9904** |

*Наилучшие результаты показала модель градиентного бустинга **XGBoost**, успешно распознающая 88.24% аварийных ситуаций.*

## 🛠️ Установка и запуск проекта

### 1. Клонирование репозитория
```bash
git clone https://github.com/Ntxfloy/predictive_maintenance_project.git
cd predictive_maintenance_project
```

### 2. Установка зависимостей
Убедитесь, что у вас установлен Python версии 3.9 и выше. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### 3. Запуск Streamlit-приложения
```bash
streamlit run app.py
```
